Karyotype Report Generator — User Guide

Anderson Diagnostics  |  Peripheral Blood Karyotyping (PBK)

1. Quick Start

  • Single Entry — Select a Report Type, click Apply Template, fill in patient details, then click Generate Report.
  • Bulk Upload — Load an Excel file (optionally with a folder of karyogram images), review / edit each patient in the inline editor, then click Generate All.

2. Report Types & Templates

Auto-detection: When loading from Excel the Report Type is automatically guessed from the ISCN result. You can change it in the editor and click Apply Template to update Interpretation / Comments / Recommendations.

3. Layout Rules

  • 2-page — Comments field is empty (Normal Male / Female).
  • 3-page — Comments field has content (all abnormal / variant reports).

4. Image Auto-Discovery (Bulk)

Select a folder of karyogram images alongside the Excel file. Files are matched by Sample Number:

  • 260154818.jpg — single image (exact match)
  • 260161295 1.jpg, 260161295 2.jpg — multiple images (numbered suffix)

Use Add Image(s) in the editor to manually adjust if auto-discovery misses files.
