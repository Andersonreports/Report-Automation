HLA Typing Report Generator

Automated HLA NGS typing report generation — Anderson Diagnostic Services

Quick Start — Bulk Upload

  1. Go to Bulk Upload tab
  2. Browse and select your Excel file (HLA typing, Luminex, CDC, DSA, Flow or PRA)
  3. Tick NABL-Accredited for MINISEQ files; untick for SURFSEQ
  4. The file is parsed automatically — report type is auto-detected from the file name and sheet contents
  5. Select a case from the list to review and edit it in the editor
  6. Select output folder and click Generate All Reports or Generate Selected

SheetPurpose

MINISEQ → enable NABL stamp. SURFSEQ → disable NABL stamp. Crossmatch, Luminex and PRA templates use their own sheet layouts — the parser locates the right sheet by its column headers.

Manual Entry

Choose the Template at the top — the form adapts to the selected report type. For HLA typing, fill the Patient Information form and enter alleles in the HLA Results section (one row per locus); enable Donor Information for transplant or RPL cases.

Signatures

  • Each report template has its own signature count (default: Single HLA 3, all others 2)
  • Configure per-template counts and signatory names in the Settings tab
